# Migration

Using the GX+ Migration Plugin, you can easily migrate your existing Gen X configuration to GX+ with just the click of a button
:::note
While GX+ migrates *most* of your configuration, some configuration opens, such as certain timings, may need to be manually converted over
:::

GX+ also comes with a lot more configuration options that will allow you to customize your experience even further, refer to the [detailed configuration](/docs/configuration/detailed) for more information
